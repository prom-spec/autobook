# Privacy Policy — OpenLoud

**Last updated:** April 6, 2026

## Overview
OpenLoud is an offline audiobook reader that converts ebook files to speech using text-to-speech (TTS) engines. Your privacy is important to us.

## Data Collection
OpenLoud does **not** collect, store, or transmit any personal data. Specifically:

- **No analytics or tracking** — we don't use any analytics SDKs
- **No user accounts** — no registration or sign-in required
- **No data leaves your device** — all book processing happens locally

## Edge TTS (Optional)
If you enable the Microsoft Edge TTS engine, text from your books is sent to Microsoft's TTS servers to generate speech audio. This is the **only** network connection the app makes. You can avoid this entirely by using the built-in Android system TTS engine.

## Storage
OpenLoud stores your imported books and settings locally on your device. This data is never transmitted anywhere.

## Permissions
- **Storage**: To read ebook files you import
- **Internet**: Only used when Edge TTS engine is selected
- **Foreground Service**: To continue playback when the app is in the background

We may update this policy. Changes will be posted in this document with an updated date.
